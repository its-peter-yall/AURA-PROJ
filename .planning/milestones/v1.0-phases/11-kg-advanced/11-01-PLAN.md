# 11-01-PLAN: Auto Summarization

<objective>
Add automatic summarization capabilities for notes and modules in AURA-NOTES-MANAGER.

Purpose: Generate concise summaries at document and module levels using LLM, providing staff with quick overviews of content and enabling students to grasp key concepts efficiently.

Output: Summarization service with API endpoints for note-level and module-level summaries.
</objective>

- `_get_cached_summary(self, key: str) -> Optional[Summary]`

Models:
```python
class SummaryLength(str, Enum):
    BRIEF = "brief"           # ~100 words
    STANDARD = "standard"     # ~250 words
    DETAILED = "detailed"     # ~500 words

class DocumentSummary(BaseModel):
    document_id: str
    document_title: str
    summary: str
    key_entities: List[str]
    key_concepts: List[str]
    word_count: int
    generated_at: datetime
    cache_key: str

class ModuleSummary(BaseModel):
    module_id: str
    module_name: str
    summary: str
    document_count: int
    document_summaries: List[DocumentSummary]
    key_themes: List[str]
    entity_frequency: Dict[str, int]
    generated_at: datetime
```

Summary prompt template:
```
You are summarizing academic content for a learning platform.

Content to summarize:
{content}

Key entities from knowledge graph:
{entities}

Instructions:
1. Create a {length} summary (~{word_count} words)
2. Highlight the main concepts and themes
3. Reference key entities where relevant
4. Use clear, academic language
5. Structure with logical flow

Format:
SUMMARY:
<your summary>

KEY_POINTS:
- Point 1
- Point 2
- Point 3

KEY_ENTITIES:
- Entity 1: brief relevance
- Entity 2: brief relevance
```

Module-level aggregation:
1. Get all documents in module
2. Generate document-level summaries (or retrieve cached)
3. Aggregate summaries with entity frequency analysis
4. Generate module-level synthesis

Caching strategy:
- Cache key: `summary:{type}:{id}:{length}:{content_hash}`
- Store in Redis with 24-hour TTL
- Invalidate on document update

<action>
Create API router for summary endpoints.

Purpose: REST endpoints for generating and retrieving summaries.

Router:
```python
router = APIRouter(prefix="/v1/summaries", tags=["Summaries"])
```

Endpoints:
```python
@router.post("/document/{document_id}", response_model=DocumentSummary)
async def summarize_document(
    document_id: str,
    length: SummaryLength = Query(SummaryLength.STANDARD),
    force_regenerate: bool = Query(False),
    summary_service: SummaryService = Depends(get_summary_service)
) -> DocumentSummary:
    """Generate or retrieve cached summary for a document."""

@router.post("/module/{module_id}", response_model=ModuleSummary)
async def summarize_module(
    module_id: str,
    length: SummaryLength = Query(SummaryLength.STANDARD),
    include_document_summaries: bool = Query(True),
    force_regenerate: bool = Query(False),
    background_tasks: BackgroundTasks,
    summary_service: SummaryService = Depends(get_summary_service)
) -> Union[ModuleSummary, TaskStatus]:
    """
    Generate or retrieve cached summary for a module.
    For large modules, returns task_id for background processing.
    """

@router.get("/document/{document_id}", response_model=DocumentSummary)
async def get_document_summary(
    document_id: str,
    length: SummaryLength = Query(SummaryLength.STANDARD),
    summary_service: SummaryService = Depends(get_summary_service)
) -> DocumentSummary:
    """Retrieve cached summary if exists, 404 if not generated."""

@router.get("/module/{module_id}", response_model=ModuleSummary)
async def get_module_summary(
    module_id: str,
    length: SummaryLength = Query(SummaryLength.STANDARD),
    summary_service: SummaryService = Depends(get_summary_service)
) -> ModuleSummary:
    """Retrieve cached module summary if exists."""

@router.delete("/document/{document_id}")
async def invalidate_document_summary(
    document_id: str,
    summary_service: SummaryService = Depends(get_summary_service)
) -> Dict[str, str]:
    """Invalidate cached summary for a document."""
```
