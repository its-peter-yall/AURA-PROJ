Models:
```python
class DocumentContext(BaseModel):
    document_id: str
    document_title: str
    module_id: str
    chunks: List[Dict[str, Any]]
    entities: List[str]
    relevance_score: float

class Citation(BaseModel):
    reference_id: str
    document_id: str
    document_title: str
    chunk_id: str
    chunk_text: str
    position: int  # Position in answer where citation applies

class ContradictionInfo(BaseModel):
    topic: str
    source_a: str
    statement_a: str
    source_b: str
    statement_b: str
    resolution_hint: Optional[str] = None

class SynthesizedAnswer(BaseModel):
    answer: str
    confidence: float
    sources_used: int
    key_points: List[str]
    citations: List[Citation]
    contradictions: List[ContradictionInfo]

class MultiDocOptions(BaseModel):
    max_documents: int = 10
    max_chunks_per_document: int = 5
    include_entity_context: bool = True
    detect_contradictions: bool = True
    citation_style: Literal["inline", "footnote", "reference"] = "inline"

class MultiDocResponse(BaseModel):
    query: str
    answer: SynthesizedAnswer
    documents_searched: int
    documents_used: int
    processing_time_ms: float
    module_ids: List[str]
```

Context gathering approach:
1. Run hybrid search across all documents in module_ids
2. Group results by document_id
3. For each document, get top N most relevant chunks
4. Include entity context from graph for each document
5. Return structured DocumentContext list

Answer synthesis approach:
1. Build prompt with all gathered contexts
2. Use Gemini to synthesize coherent answer
3. Request inline citations in the response
4. Post-process to extract and validate citations

Prompt for synthesis:
```
You are analyzing multiple documents to answer a question.

Question: {query}

Documents and relevant excerpts:
{formatted_contexts}

Instructions:
1. Synthesize a comprehensive answer using information from multiple sources
2. Use [1], [2], etc. to cite sources inline
3. If sources contradict, note the contradiction and explain
4. List key points at the end
5. Rate your confidence (0.0-1.0) based on source quality and agreement

Format:
ANSWER: <your synthesized answer with [N] citations>
KEY_POINTS:
- Point 1
- Point 2
CONFIDENCE: <0.0-1.0>
CONTRADICTIONS: <list any contradictions found, or "None">
```
