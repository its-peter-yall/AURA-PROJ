<action>
Create FastAPI router for query endpoints.

Purpose: REST API for knowledge graph search and analysis.

Router setup:
```python
router = APIRouter(prefix="/v1/kg", tags=["Knowledge Graph"])
```

Endpoints:

1. `POST /v1/kg/query` - Main search endpoint
```python
@router.post("/query", response_model=SearchResponse)
async def search_knowledge_graph(
    request: SearchRequest,
    rag_engine: RAGEngine = Depends(get_rag_engine)
) -> SearchResponse:
    """
    Perform hybrid search with optional graph expansion.
    
    - Vector + fulltext weighted search
    - Optional query expansion
    - Optional graph traversal for related entities
    - Module filtering
    """
```

2. `POST /v1/kg/analyze` - Analysis endpoint
```python
@router.post("/analyze", response_model=AnalysisResponse)
async def analyze_content(
    request: AnalysisRequest,
    rag_engine: RAGEngine = Depends(get_rag_engine)
) -> AnalysisResponse:
    """
    Perform analysis operations on chunks or documents.
    
    Operations:
    - summarize: Generate summary of content
    - compare: Compare two chunks/documents
    - extract: Extract specific information (entities, key points)
    - explain: Explain a concept using graph context
    """
```

3. `GET /v1/kg/graph/schema` - Schema endpoint
```python
@router.get("/graph/schema", response_model=GraphSchema)
async def get_graph_schema(
    graph_manager: GraphManager = Depends(get_graph_manager)
) -> GraphSchema:
    """
    Get knowledge graph schema.
    
    Returns node types, relationship types, and property definitions.
    """
```

4. `GET /v1/kg/graph/data` - Graph data endpoint
```python
@router.get("/graph/data", response_model=GraphData)
async def get_graph_data(
    module_id: Optional[str] = None,
    entity_types: Optional[List[str]] = Query(None),
    limit: int = Query(100, ge=1, le=1000),
    graph_manager: GraphManager = Depends(get_graph_manager)
) -> GraphData:
    """
    Get graph data for visualization.
    
    Returns nodes and edges for frontend graph rendering.
    """
```

Dependency injection:
```python
async def get_rag_engine() -> RAGEngine:
    driver = await get_neo4j_driver()
    embedding_service = EmbeddingService()
    return RAGEngine(driver, embedding_service)

async def get_graph_manager() -> GraphManager:
    driver = await get_neo4j_driver()
    return GraphManager(driver)
```

Error handling:
- 400: Invalid request parameters
- 404: Module/document not found
- 500: Internal server error
- 503: Neo4j connection error

<action>
Create Pydantic schemas for analysis requests and responses.

Purpose: Define API contracts for the analyze endpoint.

Schemas:
```python
class AnalysisOperation(str, Enum):
    SUMMARIZE = "summarize"
    COMPARE = "compare"
    EXTRACT = "extract"
    EXPLAIN = "explain"

class AnalysisRequest(BaseModel):
    operation: AnalysisOperation
    target_ids: List[str]  # chunk_ids, document_ids, or entity_ids
    target_type: Literal["chunk", "document", "entity"] = "chunk"
    module_ids: Optional[List[str]] = None
    options: Dict[str, Any] = {}
    # Options by operation:
    # summarize: max_length, style (brief/detailed)
    # compare: comparison_aspects
    # extract: extraction_type (entities/key_points/definitions)
    # explain: depth (basic/detailed/comprehensive)

class SummaryResult(BaseModel):
    summary: str
    key_points: List[str]
    source_ids: List[str]

class ComparisonResult(BaseModel):
    similarities: List[str]
    differences: List[str]
    source_a: str
    source_b: str

class ExtractionResult(BaseModel):
    extracted_items: List[Dict[str, Any]]
    extraction_type: str
    source_ids: List[str]

class ExplanationResult(BaseModel):
    explanation: str
    related_concepts: List[str]
    graph_context: Dict[str, Any]

class AnalysisResponse(BaseModel):
    operation: AnalysisOperation
    result: Union[SummaryResult, ComparisonResult, ExtractionResult, ExplanationResult]
    processing_time_ms: float
    model_used: str
```

```python
class NodeTypeSchema(BaseModel):
    name: str
    properties: List[str]
    count: int
    has_embedding: bool

class RelationshipTypeSchema(BaseModel):
    name: str
    source_types: List[str]
    target_types: List[str]
    properties: List[str]
    count: int

class GraphSchema(BaseModel):
    node_types: List[NodeTypeSchema]
    relationship_types: List[RelationshipTypeSchema]
    total_nodes: int
    total_relationships: int
    last_updated: datetime

class GraphNode(BaseModel):
    id: str
    label: str  # Node type
    name: str
    properties: Dict[str, Any]
    x: Optional[float] = None  # For pre-computed layouts
    y: Optional[float] = None

class GraphEdge(BaseModel):
    id: str
    source: str
    target: str
    type: str
    properties: Dict[str, Any]

class GraphData(BaseModel):
    nodes: List[GraphNode]
    edges: List[GraphEdge]
    node_count: int
    edge_count: int
    module_id: Optional[str] = None
```
