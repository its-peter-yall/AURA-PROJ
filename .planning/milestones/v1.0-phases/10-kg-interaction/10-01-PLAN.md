```python
class SearchResult(BaseModel):
    id: str
    node_type: Literal["Chunk", "ParentChunk", "Entity"]
    text: str
    score: float
    vector_score: Optional[float] = None
    fulltext_score: Optional[float] = None
    document_id: str
    module_id: Optional[str] = None
    metadata: Dict[str, Any] = {}

class SearchResults(BaseModel):
    query: str
    results: List[SearchResult]
    total_count: int
    search_time_ms: float
    weights: Dict[str, float]
```

Vector search Cypher:
```cypher
CALL db.index.vector.queryNodes('chunk_vector_index', $top_k, $query_embedding)
YIELD node, score
WHERE node.module_id IN $module_ids OR $module_ids IS NULL
RETURN node.id as id, 'Chunk' as node_type, node.text as text, score,
       node.document_id as document_id, node.module_id as module_id
ORDER BY score DESC
LIMIT $top_k
```

Fulltext search Cypher:
```cypher
CALL db.index.fulltext.queryNodes('chunk_fulltext_index', $query)
YIELD node, score
WHERE node.module_id IN $module_ids OR $module_ids IS NULL
RETURN node.id as id, 'Chunk' as node_type, node.text as text, score,
       node.document_id as document_id, node.module_id as module_id
ORDER BY score DESC
LIMIT $top_k
```

Result combination algorithm:
1. Normalize scores to 0-1 range
2. Apply weights: combined_score = (vector_score * 0.7) + (fulltext_score * 0.3)
3. Handle results appearing in only one search (use 0 for missing)
4. Sort by combined score, deduplicate by id

Include file header per project standards.
