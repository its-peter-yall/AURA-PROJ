Class method `from_env()` that populates all sub-configs and reads `AURA_TEST_MODE`.

**providers/base.py:** `AURA_EMBEDDING_DIMENSIONS = 768` module constant. `BaseProvider(ABC)` with abstract methods: `async def generate(self, request: GenerateRequest) -> GenerateResponse`, `async def stream(self, request: GenerateRequest) -> AsyncGenerator[StreamChunk, None]`, `async def list_models(self) -> list[ModelInfo]`, `async def health_check(self) -> bool`. `BaseEmbeddingProvider(ABC)` with abstract `async def _embed_raw(self, texts: list[str]) -> list[list[float]]` and concrete `async def embed(self, texts: list[str]) -> list[list[float]]` that calls `_embed_raw`, then validates every vector length == `AURA_EMBEDDING_DIMENSIONS`, raising `EmbeddingDimensionError` on mismatch. Also add concrete `async def embed_single(self, text: str) -> list[float]` that delegates to `embed([text])[0]`.
