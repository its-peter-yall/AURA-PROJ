# Plan 09-03: General Compute Provider — Frontend Settings UI (Both Apps)

Adds General Compute to the frontend settings UI in both AURA-CHAT and AURA-NOTES-MANAGER: type definitions, provider status cards, and API key management.

## must_haves

- Both `types/settings.ts` files include `'general_compute'` in the `ProviderType` union
- Both `ProviderSettingsSection.tsx` files show a General Compute card with `Cloud` icon and `needsKey: true`
- Both `ApiKeyManager.tsx` files include a General Compute entry for API key management
- Both `ProviderSettingsSection.tsx` grids use `lg:grid-cols-4` (was `sm:grid-cols-3`) to accommodate 4 providers
- Both `ApiKeyManager.tsx` grids use `xl:grid-cols-3` to accommodate 3 key-bearing providers
- Both AURA-CHAT and AURA-NOTES-MANAGER frontends build without TypeScript errors
