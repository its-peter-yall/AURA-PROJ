2. Add HAS_PARENT_CHUNK relationship from Document to ParentChunk
3. Add HAS_CHILD relationship from ParentChunk to Chunk
4. Create vector indices for all entity types (Topic, Concept, Methodology, Finding) with 768 dimensions
5. Create vector index for ParentChunk embeddings
6. Create fulltext index on Chunk.text for hybrid search
7. Add entity-entity relationship types: DEFINES, DEPENDS_ON, USES, SUPPORTS, CONTRADICTS, EXTENDS, IMPLEMENTS, REFERENCES, RELATED_TO
8. All indices must use cosine similarity function
</requirements>
