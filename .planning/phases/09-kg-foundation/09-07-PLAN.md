- Convert table to markdown-like format
- Preserve column/row structure

```python
def _table_to_text(self, table: Table) -> str:
    rows = []
    for row in table.rows:
        cells = [cell.text.strip() for cell in row.cells]
        rows.append(" | ".join(cells))
    return "\n".join(rows)
```
