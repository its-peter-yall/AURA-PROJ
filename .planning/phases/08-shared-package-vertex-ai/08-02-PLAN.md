**router.py:** Create `ModelRouter` class.

`__init__(self, config: RouterConfig | None = None)`: Store config (default `RouterConfig()`). Initialize `_providers: dict[ProviderType, BaseProvider] = {}` and `_embedding_provider: BaseEmbeddingProvider | None = None`. If config is provided and not empty, auto-register: if test mode or config.vertex_ai.project_id, create `VertexAIProvider(config.vertex_ai)` and `VertexAIEmbeddingProvider(config.vertex_ai)`, register both.

`register_provider(self, provider_type: ProviderType, provider: BaseProvider)`: Add to `_providers`.

`register_embedding_provider(self, provider: BaseEmbeddingProvider)`: Set `_embedding_provider`.

`_resolve_provider(self, request: GenerateRequest) -> BaseProvider`: If `request.provider` is set and exists in `_providers`, return it. Elif `"/" in request.model`, try `ProviderType.OPENROUTER`. Else try `self._config.default_provider`. If not found, raise `ModelUnavailableError(f"No provider registered for {resolved_type}", provider=str(resolved_type), model=request.model)`.

`async def generate(self, request: GenerateRequest | None = None, **kwargs) -> GenerateResponse`: If request is None and kwargs provided, build `GenerateRequest(**kwargs)`. Resolve provider. Call `provider.generate(request)`. Return response.

`async def embed(self, texts: list[str] | None = None, *, text: str | None = None) -> list[list[float]]`: If `text` provided and `texts` is None, set `texts = [text]`. If no `_embedding_provider`, raise `ModelRouterError("No embedding provider registered")`. Call `self._embedding_provider.embed(texts)`. Return result. (Dimension validation happens in `BaseEmbeddingProvider.embed()`.)

`async def stream(self, request: GenerateRequest | None = None, **kwargs) -> AsyncGenerator[StreamChunk, None]`: Resolve provider. Delegate to `provider.stream(request)`. Yield chunks.

Module-level `_default_router: ModelRouter | None = None` and `get_default_router() -> ModelRouter`: If `_default_router` is None, create from `RouterConfig.from_env()`, store in `_default_router`. Return it. Also add `reset_default_router()` for testing.
