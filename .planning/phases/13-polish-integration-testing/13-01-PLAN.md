**File 2: `tests/test_router_performance.py`**

Router overhead benchmark using `time.perf_counter_ns()`.

1. `test_router_generate_overhead_under_10ms`: Create test-mode router. Get direct provider via `router._providers[ProviderType.VERTEX_AI]`. Run 200 iterations of `provider.generate(request)` to get `direct_avg_ns`. Run 200 iterations of `router.generate(request=request)` to get `router_avg_ns`. Calculate `overhead_ms = (router_avg_ns - direct_avg_ns) / 1_000_000`. Assert `overhead_ms < 10.0`.

2. `test_router_stream_overhead_under_10ms`: Same pattern but for `stream()`. Consume all chunks in each iteration. Assert overhead < 10ms.

3. `test_provider_resolution_overhead_under_1ms`: Measure just `router._resolve_provider(request)` 1000 times. Assert < 1ms average (routing logic itself should be sub-millisecond).
