Classes:

1. `ExtractionTemplate` - Base template definition
```python
class ExtractionTemplate(BaseModel):
    id: str
    name: str
    description: str
    document_patterns: List[str]  # Regex patterns to detect this type
    sections: List[SectionTemplate]
    entity_types: List[str]  # Entity types to extract
    relationship_types: List[str]  # Relationship types to extract
    extraction_prompts: Dict[str, str]  # Section-specific prompts
    is_builtin: bool = True
    created_by: Optional[str] = None

class SectionTemplate(BaseModel):
    name: str
    optional: bool = False
    patterns: List[str]  # Patterns to identify section
    entity_focus: List[str]  # Primary entity types for this section
```

2. `TemplateRegistry` - Manage templates
```python
class TemplateRegistry:
    def __init__(self):
        self._templates: Dict[str, ExtractionTemplate] = {}
        self._load_builtin_templates()
    
    def register(self, template: ExtractionTemplate) -> None
    def get(self, template_id: str) -> ExtractionTemplate
    def list_all(self) -> List[ExtractionTemplate]
    def detect_template(self, content: str) -> Tuple[ExtractionTemplate, float]
    def _load_builtin_templates(self) -> None
```

3. `TemplateExtractor` - Apply templates
```python
class TemplateExtractor:
    def __init__(self, llm_client, registry: TemplateRegistry)
    
    def extract_with_template(
        self, 
        content: str, 
        template_id: str = None,  # Auto-detect if None
        options: ExtractionOptions = None
    ) -> TemplateExtractionResult
    
    def _detect_sections(self, content: str, template: ExtractionTemplate) -> List[DetectedSection]
    def _extract_section(self, section: DetectedSection, template: SectionTemplate) -> SectionExtractionResult
    def _compute_quality_score(self, result: TemplateExtractionResult, template: ExtractionTemplate) -> float
```

Built-in templates:

**Lecture Notes Template:**
- Sections: Introduction, Main Topics, Key Concepts, Examples, Summary
- Entity focus: Topic, Concept, Definition
- Relationships: DEFINES, EXPLAINS, ILLUSTRATES

**Research Paper Template:**
- Sections: Abstract, Introduction, Methodology, Results, Discussion, Conclusion
- Entity focus: Methodology, Finding, Concept
- Relationships: USES, SUPPORTS, CONTRADICTS, EXTENDS

**Meeting Notes Template:**
- Sections: Attendees, Agenda, Discussion, Action Items, Decisions
- Entity focus: Topic, Decision, Action
- Relationships: ASSIGNED_TO, DEPENDS_ON, FOLLOWS

**Lab Report Template:**
- Sections: Objective, Materials, Procedure, Observations, Analysis, Conclusion
- Entity focus: Methodology, Finding, Concept
- Relationships: USES, PRODUCES, VALIDATES

**Case Study Template:**
- Sections: Background, Problem, Analysis, Solution, Outcome, Lessons
- Entity focus: Concept, Methodology, Finding
- Relationships: APPLIES, SOLVES, DEMONSTRATES

Template detection:
- Analyze document structure (headings, patterns)
- Match against template patterns
- Return best-matching template with confidence score

```python
router = APIRouter(prefix="/v1/templates", tags=["Templates"])
```

Endpoints:
```python
@router.get("/", response_model=List[ExtractionTemplate])
async def list_templates(
    include_custom: bool = Query(True),
    template_registry: TemplateRegistry = Depends(get_template_registry)
) -> List[ExtractionTemplate]:
    """List all available extraction templates."""

@router.get("/{template_id}", response_model=ExtractionTemplate)
async def get_template(
    template_id: str,
    template_registry: TemplateRegistry = Depends(get_template_registry)
) -> ExtractionTemplate:
    """Get a specific template by ID."""

@router.post("/", response_model=ExtractionTemplate)
async def create_template(
    template: ExtractionTemplate,
    template_registry: TemplateRegistry = Depends(get_template_registry)
) -> ExtractionTemplate:
    """Create a custom extraction template."""

@router.post("/detect", response_model=TemplateDetectionResult)
async def detect_template(
    content: str = Body(...),
    template_registry: TemplateRegistry = Depends(get_template_registry)
) -> TemplateDetectionResult:
    """Detect the best matching template for content."""

@router.post("/preview", response_model=TemplatePreviewResult)
async def preview_extraction(
    content: str = Body(...),
    template_id: str = Body(...),
    template_extractor: TemplateExtractor = Depends(get_template_extractor)
) -> TemplatePreviewResult:
    """Preview extraction results without saving."""
```

Models:
