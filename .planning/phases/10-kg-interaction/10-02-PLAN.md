GraphContext model:
```python
class EntityPath(BaseModel):
    source_entity: str
    target_entity: str
    relationship_type: str
    confidence: float
    hops: int

class GraphContext(BaseModel):
    seed_entities: List[str]
    expanded_entities: List[Dict[str, Any]]
    paths: List[EntityPath]
    total_entities: int
    max_depth_reached: int
```

Traversal Cypher (2-hop):
```cypher
MATCH path = (start:Entity)-[r1]->(e1:Entity)-[r2]->(e2:Entity)
WHERE start.id IN $entity_ids
AND (start.module_id IN $module_ids OR $module_ids IS NULL)
AND (e1.module_id IN $module_ids OR $module_ids IS NULL)
AND (e2.module_id IN $module_ids OR $module_ids IS NULL)
RETURN 
    start.name as source,
    e1.name as intermediate,
    e2.name as target,
    type(r1) as rel1_type,
    type(r2) as rel2_type,
    r1.confidence as rel1_confidence,
    r2.confidence as rel2_confidence,
    length(path) as hops
```

Also support 1-hop queries:
```cypher
MATCH (start:Entity)-[r]->(related:Entity)
WHERE start.id IN $entity_ids
AND (related.module_id IN $module_ids OR $module_ids IS NULL)
RETURN start.name as source, related.name as target,
       type(r) as relationship_type, r.confidence as confidence
```

Relationship weights (for ranking):
- DEFINES: 1.0 (strongest semantic connection)
- DEPENDS_ON: 0.9
- USES: 0.8
- SUPPORTS: 0.8
- EXTENDS: 0.7
- IMPLEMENTS: 0.7
- CONTRADICTS: 0.6
- REFERENCES: 0.5
- RELATED_TO: 0.4
