```bash

import asyncio
import os
os.environ['AURA_TEST_MODE'] = 'true'  # Use test mode without credentials
from services.llm_entity_extractor import extract_entities, merge_extraction_results

async def test():
    sample_text = '''Machine learning is a subset of artificial intelligence.
    Neural networks are computing systems inspired by biological neural networks.
    Researchers at Google have contributed significantly.'''

    result = await extract_entities(sample_text, 'test-chunk')
    
    # result.entities is a dict keyed by category: concepts, topics, methodologies, findings
    total = sum(len(v) for v in result.entities.values())
    print(f'Extracted {total} entities:')
    for category, entities in result.entities.items():
        for e in entities:
            print(f'  - {e.name} ({e.type}) [{category}]')
    
    print(f'Extracted {len(result.relationships)} relationships')

asyncio.run(test())
"
```

### Additions Made
1. **Module-level convenience functions** added at end of file:
   - `extract_entities(text, chunk_id)` → `ExtractionResult`
   - `merge_extraction_results(results)` → merged entities dict

2. **ExtractionResult dataclass** added:
   - `entities: Dict[str, List[ExtractedEntity]]`
   - `relationships: List[Relationship]`
   - `raw_response: Optional[str]`

3. **Circular import fix**: Changed `from api.config import LLM_ENTITY_EXTRACTION_MODEL` to direct `os.getenv()` to break circular dependency with `api` package.

### Key Components
- `LLMEntityExtractor` class with async batch processing
- `ExtractedEntity` Pydantic model (name, type, definition, confidence_score, source_text, category)
- `Relationship` Pydantic model (source_entity, target_entity, relationship_type, confidence, evidence)
- XML-structured prompts (`ENTITY_EXTRACTION_PROMPT`)
- Test mode support via `AURA_TEST_MODE=true`
- Retry logic with exponential backoff
